# Media Quick Downloader (Firefox)

## How to load locally for testing
1. Save the files to a folder, e.g. `media-downloader/`.
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`.
3. Click **"Load Temporary Add-on"** and select the `manifest.json` from the folder.
4. Visit a page with a playable video and look for a small ⬇︎ button on the top-right of the media element.
